AdBlock Origin

The AdBlock goes to its original roots!

The aim of this project is to make AdBlock by people, for people once again.

What is AdBlock Origin?

AdBlock Origin is a fork of AdBlock, which is still in development by the AdBlock Team.


AdBlock Origin doesn't enable Acceptable Ads by default or send any info back to the devs without your permission, unlike AdBlock.


Like AdBlock before us, AdBlock Origin is 100% open-source, licensed under the GNU GPL v3. Feel free to check out the code at GitHub.

What goals do we have?

We've determined some goals we want to fulfill.

The current goals of this project are:

  • be more transparent and open about the upcoming changes
  • be an independent extension, which is not controlled by ad companies
  • be less resource intrusive than AdBlock
  • be working for the user, not for money
  • be modern, using the latest browser technology
